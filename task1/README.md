# Lab 9.1 — Оптимизация React компонентов

## Описание

В данной работе были изучены методы оптимизации React-приложений:

- React.memo
- useMemo
- useCallback

## Реализация

### 1. React.memo

Использован в компоненте UserCard для предотвращения лишних ререндеров, если props не изменились.

### 2. useMemo

Использован в AnalyticsChart для кэширования результата тяжелых вычислений.

### 3. useCallback

Использован для стабилизации функций и предотвращения лишнего создания функций при рендере.

## Результат

После оптимизации количество лишних ререндеров уменьшилось, что видно в консоли браузера.
