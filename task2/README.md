# Lab 9.2 — Виртуализация списков

## Описание

В данной работе реализована виртуализация списка для оптимизации отображения большого количества элементов.

## Реализация

### 1. Генерация данных

Создано 10000 элементов с различными свойствами.

### 2. RegularList

Обычный список, который рендерит все элементы сразу.

### 3. VirtualList

Используется библиотека react-window для отображения только видимых элементов.

### 4. Фильтрация

Добавлена возможность фильтрации элементов по названию и категории.

## Сравнение

RegularList:

- Рендерит все элементы
- Может тормозить

VirtualList:

- Рендерит только видимые элементы
- Быстрая прокрутка
- Меньше нагрузка на DOM

## Вывод

Виртуализация значительно улучшает производительность при работе с большими списками.
